# Code of Conduct

## Our standard

Be respectful, specific, and evidence-driven.

This repository deals with destructive local-machine operations. Review feedback should focus on correctness, safety, and reproducibility.

## Unacceptable behavior

- abusive or insulting language
- encouraging unsafe destructive testing without disclosure
- hiding risky behavior in patches

## Enforcement

Maintainers may remove unsafe or hostile contributions from the project.
